# AI-Based Text & PDF Summarizer

## Project Overview

This project is an AI-powered Text and PDF Summarization web application built using React.js, FastAPI, and HuggingFace Transformers. The application allows users to enter text or upload PDF files and generates concise summaries using Natural Language Processing (NLP).

---

## Features

* Text Summarization
* PDF Summarization
* AI-powered NLP model
* Responsive React frontend
* FastAPI backend
* Copy Summary feature
* Clear button
* Word & Character Counter

---

## Installation

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

---

## API Endpoints

### Text Summarization

POST `/summarize-text`

### PDF Summarization

POST `/summarize-pdf`
